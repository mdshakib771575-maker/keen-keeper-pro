# 👥 KeenKeeper — Keep Your Friendships Alive

![KeenKeeper Cover](https://via.placeholder.com/1200x500/6366F1/FFFFFF?text=KeenKeeper+%E2%80%94+Never+Let+Friendships+Fade)

**KeenKeeper** হলো একটি সুন্দর ও সহজ ওয়েব অ্যাপ যা তোমাকে তোমার বন্ধুদের সাথে যোগাযোগ বজায় রাখতে সাহায্য করে। এখানে তুমি দেখতে পারবে কতদিন আগে কার সাথে কথা বলেছ, কন্টাক্ট গোল সেট করতে পারবে, কল/টেক্সট/ভিডিও কল লগ করতে পারবে এবং সুন্দর অ্যানালিটিক্স দেখতে পারবে।

**"ভালো বন্ধুত্ব কখনো হারিয়ে যাক না"** 💛

---

## ✨ মূল ফিচারসমূহ

- ✅ **Responsive Navbar** — লোগো + Home, Timeline, Stats (আইকনসহ + Active link highlight)
- ✅ **Hero Banner** + "Add a Friend" বাটন
- ✅ **Friend Cards** — ছবি, নাম, Days Since Contact, Tags, Status (Overdue / Almost Due / On Track) সহ
- ✅ **Friend Detail Page** — দুই কলাম লেআউট, Stats, Relationship Goal, Quick Check-In
- ✅ **Quick Check-In** — Call / Text / Video বাটনে ক্লিক করলেই Timeline-এ এন্ট্রি যোগ হয় + Toast Notification
- ✅ **Timeline Page** — সব ইন্টারেকশনের ইতিহাস (Date + Icon + Title)
- ✅ **Friendship Analytics (Stats)** — Recharts Pie Chart দিয়ে Call/Text/Video এর পরিসংখ্যান
- ✅ **Timeline Filter** — Call / Text / Video অনুযায়ী ফিল্টার
- ✅ **404 Page**, Loading Animation, Fully Responsive Design (Mobile + Tablet + Desktop)

---

## 🛠️ ব্যবহৃত টেকনোলজি

- **Frontend**: React.js + Vite
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Notifications**: React Hot Toast
- **Icons**: Lucide React
- **Data**: Local `friends.json`

## 🚀 কিভাবে রান করবেন (Local Setup)

```bash
# রিপোজিটরি ক্লোন করো
git clone https://github.com/mdshakib771575-maker/keenkeeper.git
cd keenkeeper

# ডিপেন্ডেন্সি ইনস্টল করো
npm install

# ডেভেলপমেন্ট সার্ভার চালাও
npm run dev
